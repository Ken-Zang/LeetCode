# Day1 283.移动零

[题目链接](https://leetcode.cn/problems/move-zeroes/)

依然是快慢双指针，不用真的傻傻地每次都把零移到最后，只需要把其他元素安排好再将末尾都赋值为零就好。

```java
class Solution {
    public void moveZeroes(int[] nums) {
        int slow = 0;

        for(int fast = 0; fast < nums.length; fast++){
            if(nums[fast] != 0){
                nums[slow] = nums[fast];
                slow++;
            }
        }
        for(; slow < nums.length; slow++){
            nums[slow] = 0;
        }
    }
}
```
