# Day1 367.有效的完全平方数

[题目链接](https://leetcode.cn/problems/valid-perfect-square/)

依然可以使用二分查找，与题目69不同的是，这次需要mid^2 == target才视为找到。

```java
class Solution {
    public boolean isPerfectSquare(int num) {
        int left = 0;
        int right = num;
        int mid;

        while(left <= right){
            mid = left + (right - left)/2;
            if((long)mid * mid == num){
                return true;
            } else if ((long)mid * mid < num){
                left = mid + 1;
            } else { // 这里依然可以省一次乘法
                right = mid - 1;
            }
        }

        return false;
    }
}
```
