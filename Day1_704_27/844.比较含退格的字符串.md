# Day1 844.比较含退格的字符串

[题目链接](https://leetcode.cn/problems/backspace-string-compare/)

重构字符串之后再比较的时间复杂度是O(N+M)，这种解法比较直观，可以借助栈来实现。

此题依然可以使用快慢双指针，思路是从后往前遍历字符串。此题逻辑较复杂，需要反复回顾。

```java
class Solution {
    public boolean backspaceCompare(String s, String t) {
        int sPtr = s.length() - 1;
        int tPtr = t.length() - 1;
        int skipS = 0;
        int skipT = 0;

        while(sPtr >= 0 || tPtr >= 0){
            while(sPtr >= 0){
                if(s.charAt(sPtr) == '#'){
                    skipS++;
                    sPtr--;
                } else if(skipS > 0){
                    skipS--;
                    sPtr--;
                } else {
                    break; //此时sPtr指向s实际会输出的最后一个字符
                }
            }
            while(tPtr >= 0){
                if(t.charAt(tPtr) == '#'){
                    skipT++;
                    tPtr--;
                } else if(skipT > 0){
                    skipT--;
                    tPtr--;
                } else {
                    break; //此时tPtr指向t实际会输出的最后一个字符
                }
            }
            if(sPtr >= 0 && tPtr >= 0){
                if(s.charAt(sPtr) != t.charAt(tPtr)){
                    return false;
                }
            } else if(sPtr >= 0 || tPtr >= 0){
                return false;
            }
            sPtr--;
            tPtr--;
        }
        return true;
    }
}
```
